## 说明范围

本页用于说明本站在浏览、评论、追番同步与静态资源加载过程中，可能涉及的数据处理方式。
它是面向访客的站点说明，不构成法律意见。

## 本站的基本形态

本站是一个基于 Hexo 与 AnZhiYu 主题构建的静态博客，主要由 GitHub Pages 提供托管与发布。

## 可能涉及的数据

当你访问本站时，站点托管平台、CDN、图片托管服务以及浏览器本身，可能会记录常规访问日志，例如：

- IP 地址
- 浏览器与设备信息
- 访问时间
- 引荐来源
- 请求的页面与静态资源

这些信息主要用于页面分发、故障排查、缓存控制与基础安全防护。

## 评论相关

本站当前使用 Giscus 作为评论系统，评论内容实际托管于 GitHub Discussions。
如果你在本站发表评论，相关数据会按 GitHub 与 Giscus 的规则处理，包括但不限于：

- 你的 GitHub 账号公开信息
- 评论内容与提交时间
- GitHub 侧必要的登录与安全信息

如果你不希望 GitHub 处理你的评论数据，请不要使用评论功能。

## 第三方服务

本站当前可能调用以下第三方资源或服务：

- GitHub Pages / GitHub：站点托管、评论承载
- Giscus：评论组件
- Bangumi 相关接口或公开页面：追番页同步公开条目数据
- 网易云相关资源：音乐播放入口与歌单引用
- 外部 CDN 与图床：主题静态资源、图片与脚本分发

这些服务拥有各自独立的隐私政策，本站无法替你控制第三方的处理方式。

## 浏览器存储

为了保存主题显示状态与提升页面体验，本站前端脚本可能会使用浏览器本地存储，例如：

- 深浅色模式
- 简繁切换状态
- 阅读模式或界面偏好
- 弹窗关闭状态

具体说明见 Cookies 页面。
