## 本站如何使用浏览器存储

本站是静态博客，本身不提供账号系统，也不主动设置广告或登录类 Cookie。
但主题脚本与第三方服务可能会使用 Cookie、localStorage 或 sessionStorage 来保存状态。

## 可能保存的内容

本站前端可能在你的浏览器中保存以下偏好信息：

- 深色模式或浅色模式选择
- 简繁转换状态
- 阅读模式、侧边栏、控制台等界面状态
- 隐私弹窗是否已经关闭
- 部分评论或页面缓存

这些内容主要用于提升访问体验，不会被我直接手动读取。

## 第三方可能设置的 Cookie

如果你使用或加载了以下功能，对应服务可能会设置它们自己的 Cookie 或类似标识：

- GitHub / Giscus 评论
- CDN 静态资源分发
- 第三方图片、音乐或番剧相关资源

这部分行为由第三方服务各自控制。

## 你可以怎么管理

你可以随时通过浏览器设置进行管理，例如：

- 清除本站相关的 Cookie 与站点数据
- 禁用第三方 Cookie
- 使用隐私模式访问
- 禁止某些脚本或第三方资源加载

清除浏览器站点数据后，本站的主题显示偏好也会一并重置。
